# Student Performance Prediction System

## Overview

This project uses Machine Learning (Linear Regression) to predict student marks based on the number of hours studied.

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Scikit-learn

## Project Workflow

1. Create and load dataset
2. Train a Linear Regression model
3. Predict student marks
4. Visualize results using a graph

## Results

The model successfully predicts marks based on study hours and displays the relationship using a regression line.
